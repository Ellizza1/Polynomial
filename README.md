# Проект: Polynomial & Newton Interpolation

Этот проект представляет собой реализацию лабораторной работы №1 по дисциплине "Программирование на Java". Программа предназначена для работы с полиномами произвольной степени, а также для построения интерполяционных полиномов Ньютона по набору точек.

## Описание задачи
Проект разделен на две основные логические части:

1. **Класс `Polynomial`**: базовая работа с многочленами (хранение коэффициентов, арифметические операции, приведение к строковому виду).
2. **Интерполяция Ньютона**: построение интерполяционного полинома на основе набора точек с использованием разделенных разностей.

## Основная функциональность

### `Polynomial`
* Хранение коэффициентов (List/Map).
* Несколько вариантов конструкторов (по умолчанию, из списка, из массива, переменное количество аргументов).
* Арифметические операции:
  * `plus()` — сложение полиномов.
  * `minus()` — вычитание полиномов.
  * `times()` — умножение полиномов.
* Переопределение методов `toString()`, `equals()` и `hashCode()`.
* Автоматическая корректировка коэффициентов (удаление лишних нулевых значений).
* Определение степени полинома.

### Интерполяция Ньютона
* Хранение набора точек (координат).
* Динамическое построение полинома:
  * Добавление/удаление точек с автоматической перестройкой полинома.
* Реализация алгоритма разделенных разностей для вычисления коэффициентов.
